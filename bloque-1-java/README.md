# Bloque 1 — Programación con Java

Apuntes del Bloque 1: un resumen en markdown por tema, generado al cerrar cada uno. El orden sigue los apuntes del módulo (Fernando Rodríguez Diéguez, IES Fernando Wirtz).

## Enfoque

- **Se ven todos los temas** (para tener una referencia coherente y completa), pero con **ritmo según el diagnóstico**: rápido en lo ya dominado, pausado en lo nuevo.
- Cada tema cerrado → **resumen aquí** + **tarjetas de Anki**.
- El contenido manda sobre el reloj: avanzamos por temas, no por fechas.

## Convención de archivos

`tema-NN-nombre.md` — minúsculas, sin tildes ni espacios.
Ejemplo: `tema-01-conceptos-basicos.md`
