# Atajos de teclado de Ubuntu (GNOME)

> Chuleta de referencia para Ubuntu 26.04 con el escritorio GNOME.
> La tecla **Super** es la que tiene el logo de Windows (entre `Ctrl` y `Alt`).

---

## Escritorio y navegación general

| Atajo | Qué hace |
|---|---|
| `Super` | Abre la vista de Actividades (resumen de ventanas). Empieza a escribir para buscar apps o archivos. |
| `Super` + `A` | Muestra todas las aplicaciones. |
| `Super` + `1`...`9` | Abre o cambia a la app en esa posición del dock. |
| `Super` + `L` | Bloquea la pantalla. |
| `Super` + `D` | Muestra el escritorio (minimiza todo). Púlsalo otra vez para volver. |
| `Alt` + `F2` | Abre un cuadro para ejecutar un comando rápido. |
| `Ctrl` + `Alt` + `Supr` | Cerrar sesión (puede pedir confirmación). |

---

## Gestión de ventanas

| Atajo | Qué hace |
|---|---|
| `Alt` + `Tab` | Cambia entre aplicaciones abiertas (mantén `Alt` y pulsa `Tab`). |
| `Alt` + `` ` `` | Cambia entre ventanas de la **misma** aplicación. |
| `Super` + `Flecha izquierda` / `derecha` | Ancla la ventana a la mitad izquierda o derecha de la pantalla. |
| `Super` + `Flecha arriba` | Maximiza la ventana. |
| `Super` + `Flecha abajo` | Restaura / desmaximiza la ventana. |
| `Super` + `H` | Minimiza (oculta) la ventana actual. |
| `Alt` + `F4` | Cierra la ventana actual. |
| `Alt` + `F7` | Mover la ventana con las flechas (Esc o clic para soltar). |

---

## Escritorios virtuales (espacios de trabajo)

| Atajo | Qué hace |
|---|---|
| `Super` + `Re Pág` / `Av Pág` | Cambia al escritorio de arriba / abajo. |
| `Super` + `Shift` + `Re Pág` / `Av Pág` | Mueve la ventana actual a otro escritorio. |

---

## Capturas de pantalla y grabación

| Atajo | Qué hace |
|---|---|
| `Impr Pant` (PrtSc) | Abre la herramienta de captura (elegir zona, ventana o pantalla completa; también graba vídeo). |
| `Shift` + `Impr Pant` | Captura una zona seleccionada directamente. |
| `Ctrl` + `Impr Pant` | Captura la pantalla completa al portapapeles. |

---

## Terminal

> Abre la terminal con `Ctrl` + `Alt` + `T`.

| Atajo | Qué hace |
|---|---|
| `Ctrl` + `Alt` + `T` | Abre una terminal nueva. |
| `Ctrl` + `Shift` + `T` | Abre una pestaña nueva en la terminal. |
| `Ctrl` + `Shift` + `C` | Copiar (en la terminal, el `Ctrl+C` normal NO copia). |
| `Ctrl` + `Shift` + `V` | Pegar. |
| `Tab` | Autocompleta comandos y nombres de archivo. |
| `Flecha arriba` / `abajo` | Recorre el historial de comandos. |
| `Ctrl` + `R` | Busca en el historial de comandos mientras escribes. |
| `Ctrl` + `C` | **Cancela** el comando que se está ejecutando. |
| `Ctrl` + `L` | Limpia la pantalla (equivale al comando `clear`). |
| `Ctrl` + `A` / `Ctrl` + `E` | Salta al inicio / final de la línea. |
| `Ctrl` + `U` / `Ctrl` + `K` | Borra desde el cursor hasta el inicio / final de la línea. |
| `Ctrl` + `W` | Borra la palabra anterior al cursor. |
| `Ctrl` + `D` | Cierra la sesión de la terminal. |

---

## Edición de texto (general)

| Atajo | Qué hace |
|---|---|
| `Ctrl` + `C` / `X` / `V` | Copiar / cortar / pegar. |
| `Ctrl` + `Z` / `Ctrl` + `Shift` + `Z` | Deshacer / rehacer. |
| `Ctrl` + `S` | Guardar. |
| `Ctrl` + `F` | Buscar. |
| `Ctrl` + `A` | Seleccionar todo. |
| `Ctrl` + `W` | Cierra la pestaña o ventana actual (en muchas apps). |
| `Ctrl` + `Q` | Cierra la aplicación. |

---

## Gestor de archivos (Nautilus / "Archivos")

| Atajo | Qué hace |
|---|---|
| `Ctrl` + `L` | Escribir la ruta a mano en la barra de direcciones. |
| `Ctrl` + `H` | Mostrar / ocultar archivos ocultos (los que empiezan por `.`). |
| `Ctrl` + `Shift` + `N` | Crear una carpeta nueva. |
| `F2` | Renombrar el archivo seleccionado. |
| `Supr` | Mover a la papelera. |
| `Ctrl` + `Z` | Deshacer la última acción (mover, renombrar, borrar). |

---

*Consejo: casi todos estos atajos se pueden ver y personalizar en **Configuración → Teclado → Ver y personalizar combinaciones de teclas**.*
